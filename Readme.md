# Calculator

A simple calculator app built with Streamlit that performs basic arithmetic operations.

## Features
- Addition
- Subtraction
- Multiplication
- Division (with zero-check)

## How to Run

1. Install dependencies:
```bash
pip install streamlit
```

2. Run the app:
```bash
streamlit run calculator.py
```

3. Open your browser at `http://localhost:8501`

## How to Use
1. Enter first number
2. Enter second number
3. Select operation from dropdown
4. Click "Calculate" button
5. Result will be displayed

## Requirements
- Python 3.7+
- Streamlit
